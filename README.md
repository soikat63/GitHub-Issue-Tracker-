

## 1. What is the difference between var, let and const?

JavaScript provides three ways to declare variables.

A. **var**

  - Function scoped
  - Can be redeclared and updated
  - Hoisted to the top of its scope

B. **let**

  - Block scoped
  - Can be updated
  - Cannot be redeclared in the same scope

C. **const**

  - Block scoped
  - Cannot be updated or redeclared
  - Used for values that should remain constant


## 2. What is the Spread Operator (...)?

The spread operator is used to expand elements of an array or properties of an object into another array or object. It helps to copy, merge, or manipulate data easily in modern JavaScript.


## 3. Difference between map(), filter() and forEach()?

These are array methods used to work with array data.

1. **map()**
  Creates a new array by transforming each element of the original array.

2. **filter()**
  Creates a new array containing only the elements that satisfy a specific condition.

3. **forEach()**
  Executes a function for each array element but does not return a new array.


## 4. What is an Arrow Function?

An arrow function is a shorter and more modern way to write functions in JavaScript. It provides a cleaner syntax and is widely used in modern JavaScript and frameworks like React.


## 5. What are Template Literals?

Template literals are a modern way to work with strings in JavaScript using backticks (`` ` ``). They allow embedding variables and expressions inside a string and also support multi-line strings.
